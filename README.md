# Bounce Terminal

A bouncing ball in your Termianl

### Building:
`make bounce`

### Usage:
```
./bounce <map>
./bounce <ball size> <map>
./bounce <ball width> <ball height> <map>
```

### Instructions:
Backspace to reset ball position but not velocity  
Escape to exit instead of ^C

**Map.txt:**
```
5 5
11111
10001
10S01
10001
11111
```
`1` - Wall  
`0` - Empty  
`S` - Ball start position
